
# Express Phone Pro | إكسبريس فون برو 📱

نظام متكامل واحترافي لإدارة محلات الهواتف النقالة، مصمم ليعمل بكفاءة عالية على الهواتف والحواسيب مع دعم كامل للعمل بدون إنترنت (Offline-First).

## 🚀 الميزات الرئيسية

*   **نظام نقطة البيع (POS) الذكي:**
    *   دعم كامل للباركود وكود QR عبر كاميرا الهاتف.
    *   معاينة حرارية (Thermal Preview) احترافية قبل الطباعة.
    *   التحقق التلقائي من المخزون ومنع بيع كميات غير متوفرة.
*   **إدارة المخزون المتقدمة:**
    *   فرز (Filtering) متعدد التصنيفات في آن واحد.
    *   نظام تصنيفات هرمي (مثل: قطع غيار > شاشات > سامسونج).
    *   تنبيهات ذكية للمخزون المنخفض مع نافذة تفصيلية.
    *   إمكانية تصدير قائمة النواقص بصيغة **Excel (CSV)** أو **Text**.
*   **الديون والمالية:**
    *   تتبع دقيق لديون الزبائن مع سجل تاريخي لكل فاتورة.
    *   زر "إعادة المزامنة" لتصحيح الحسابات تلقائياً بناءً على سجل الفواتير.
    *   إحصائيات حصرية لقطاع الشاشات (قيمة الشراء والبيع).
*   **الأمان والتقنية:**
    *   نظام تتبع العمليات (Audit Logs) لمعرفة من قام بكل عملية.
    *   قاعدة بيانات Firebase مع مزامنة لحظية.
    *   دعم الوضع الليلي (Dark Mode).
    *   واجهة مستخدم متجاوبة تماماً مع الهواتف الذكية.
    *   مؤشر حالة الاتصال (Online/Offline) مع مزامنة تلقائية وآمنة.

## 🛠️ المتطلبات التقنية
*   **Framework:** Next.js 15 (App Router)
*   **Database & Auth:** Firebase (Firestore & Auth)
*   **UI Components:** ShadCN UI & Tailwind CSS
*   **Icons:** Lucide React

## 📦 حل مشكلة الرفع (403 Forbidden)
إذا واجهت خطأ `403` عند محاولة الـ `push` رغم استخدام الـ Token، اتبع ما يلي:

1. **إعدادات GitHub:**
   - انتقل إلى **Settings** > **Developer settings** > **Personal access tokens**.
   - إذا كنت تستخدم **Fine-grained tokens**:
     - تأكد من اختيار المستودع `Express-Phone-Pro` تحت قائمة **Repository access**.
     - في قائمة **Permissions**، ابحث عن **Contents** واختر **Read and Write**.
   - إذا كنت تستخدم **Tokens (classic)**:
     - تأكد من تفعيل صلاحية `repo` بالكامل.

2. **أوامر الـ Terminal:**
   ```bash
   # حذف التعريف القديم
   git remote remove origin
   
   # إضافة الرابط الجديد بالـ Token الصحيح
   git remote add origin https://TOKEN_HERE@github.com/PitaDz14/Express-Phone-Pro.git
   
   # الرفع مع تعيين الفرع الرئيسي
   git push -u origin main
   ```
