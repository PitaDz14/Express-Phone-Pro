
# Express Phone Pro | إكسبريس فون برو 📱

نظام متكامل واحترافي لإدارة محلات الهواتف النقالة، مصمم ليعمل بكفاءة عالية على الهواتف والحواسيب مع دعم كامل للعمل بدون إنترنت (Offline-First).

## 🚀 الميزات الرئيسية

*   **نظام نقطة البيع (POS) الذكي:** دعم الباركود، معاينة حرارية، والتحقق من المخزون.
*   **إدارة المخزون المتقدمة:** فرز متعدد، تنبيهات النواقص، وتصدير البيانات.
*   **نظام الصلاحيات (RBAC):** تمييز كامل بين المدير (Admin) والموظف (Worker).
*   **الديون والمالية:** تتبع دقيق للديون مع ميزة إعادة المزامنة التلقائية.
*   **تتبع العمليات:** سجل كامل يوضح اسم الموظف المسؤول عن كل فاتورة.

## 💻 دليل التشغيل المحلي (على جهازك الخاص)

إذا قمت بتحميل المشروع على حاسوبك وتريد تشغيله، اتبع الخطوات التالية:

### 1. المتطلبات الأساسية
تأكد من تثبيت **Node.js** (نسخة 18 أو أحدث) من الموقع الرسمي [nodejs.org](https://nodejs.org/).

### 2. تثبيت المكتبات (Dependencies)
افتح واجهة الأوامر (Terminal/PowerShell) داخل مجلد المشروع ونفذ:
```bash
npm install
```

### 3. تشغيل وضع التطوير (Development)
لتشغيل البرنامج وتجربة التعديلات لحظياً:
```bash
npm run dev
```
سيظهر لك رابط في Terminal، غالباً ما يكون: `http://localhost:9002`

### 4. التجميع النهائي للإنتاج (Production Build)
للحصول على أفضل أداء وسرعة في المحل، يفضل تجميع المشروع وتشغيله كنسخة نهائية:
```bash
npm run build
npm start
```

## ⚙️ دليل الانتقال لمشروع Firebase جديد

إذا أردت تغيير حساب Firebase أو المشروع، اتبع الخطوات التالية:

1.  **Firebase Console:** أنشئ مشروعاً جديداً، فعل **Firestore** و **Authentication**.
2.  **تحديث الإعدادات:** استبدل القيم في ملف `src/firebase/config.ts` ببيانات الـ `firebaseConfig` الجديدة.
3.  **المستخدم الأول:** أنشئ مستخدماً في Firebase وانسخ الـ **UID**، ثم اطلبه مني لتعيينه كـ Admin.

## 📦 حل مشكلة الرفع لـ GitHub (403 Forbidden)
إذا واجهت خطأ عند الرفع: تأكد من أن الـ **Token** الخاص بك يمتلك صلاحيات **Write** على المستودع. استخدم الأوامر التالية لتحديث الرابط:
```bash
git remote remove origin
git remote add origin https://YOUR_TOKEN_HERE@github.com/PitaDz14/Express-Phone-Pro.git
git push -u origin main
```
