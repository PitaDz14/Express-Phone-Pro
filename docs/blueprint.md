# **App Name**: Express Phone Pro

## Core Features:

- مديرية المستخدمين والصلاحيات: نظام تسجيل دخول آمن (بريد إلكتروني وكلمة مرور فقط) مع أدوار محددة (مدير، عامل)، وضمان حماية المسارات (Protected routes).
- إدارة المنتجات والمخزون: إضافة، تعديل، وحذف المنتجات بتفاصيلها الأساسية (الاسم، الفئة، الكمية، أسعار الشراء والبيع، الكود الفريد) مع تنبيهات عند انخفاض المخزون.
- إدارة العملاء والديون: إنشاء وإدارة ملفات العملاء، تتبع الديون المستحقة عليهم، وتسجيل المدفوعات الجزئية والملاحظات المتعلقة بالعملاء.
- نظام الفواتير الشامل: إنشاء، عرض، وتعديل الفواتير متعددة المنتجات مع حسابات إجمالية تلقائية وتتبع حالات الدفع (مدفوع/جزئي).
- توليد أكواد QR للملصقات: توليد أكواد QR خاصة لتحديد المنتجات وطباعة ملصقات سهلة القراءة للأسعار.
- لوحات التحليلات والتقارير: عرض لوحات معلومات أساسية لتتبع المبيعات، الأرباح، وإجمالي الديون لتوفير نظرة عامة سريعة على أداء المحل.
- أداة اقتراح وصف المنتجات بالذكاء الاصطناعي: أداة تعتمد على الذكاء الاصطناعي لتقديم اقتراحات لأوصاف جذابة للمنتجات الجديدة بناءً على الكلمات المفتاحية المدخلة.

## Style Guidelines:

- نظام الألوان سيعتمد على مخطط فاتح ليعكس النظافة والاحترافية.
- اللون الأساسي: أزرق احترافي (#3960AC) يمثل الثقة والابتكار.
- اللون الخلفي: أزرق-رمادي خفيف جداً (#F0F2F4) لخلق مساحة نظيفة وهادئة.
- لون التمييز: تركوازي حيوي (#3CC2DD) لإبراز الأزرار التفاعلية وعناصر التركيز.
- الخط الرئيسي للعناوين والنصوص هو 'Inter' (sans-serif) لضمان مظهر عصري، حيادي ووضوح مثالي للواجهة العربية ومحتوى لوحة القيادة.
- استخدام أيقونات Material Design 3، بنمط مخطط ومتسق، لتسهيل التعرف البصري وتعزيز التجربة البديهية.
- تصميم تخطيط لوحة قيادة SaaS حديثة (جهة اليمين لـRTL) مع شريط جانبي للتنقل ومحتوى رئيسي واسع، مع ضمان الاستجابة التامة لمختلف أحجام الشاشات.
- إدماج حركات انتقالية بسيطة وسريعة لتوفير تجربة مستخدم سلسة واستجابة فورية.
